This is our professional portfolio — a single web page that tells you:

- **What we built** (13 real projects across AI, Data, Software, and Fintech)
- **How to reach us** (LinkedIn, GitHub, and Email)

## How To Open It

You do **not** need to install anything.

1. Download or clone this project to your computer
2. Find the file called `index.html`
3. Double-click it — it will open in your browser automatically

That's it! ✅
